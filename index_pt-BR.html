Um plugin jQuery para editar, ler e gerenciar cookies

English version of this page →

O que é?

O CooQuery é um plugin pra jQuery que ajuda no trabalho de gerenciar, editar, criar e deletar cookies.

Funciona em quais navegadores?

CooQuery é compatível com a maioria dos navegadores modernos, e também o Internet Explorer 6:

  • Firefox
  • Opera
  • Google Chrome
  • Internet Explorer (6+)

Exemplos:

  1. Setar um novo cookie, chamado "test cookie", com o valor "This is a test cookie":

    $('#example1').click(function(){
  var cookie = $.setCookie('test cookie', 'This is a test cookie', {
    duration: 10 // in days
  });
  if(cookie)
    alert('Cookie criado!');
  else
    alert('Ops, ocorreu um erro.');
});

  2. Ler o valor do cookie "test cookie":

    $('#example2').click(function(){
  var value = $.readCookie('test cookie');
  if( value )
    alert( "Hey! O cookie existe, e seu valor é: \n" + value );
  else
    alert( 'O cookie \'non ecziste\'!' );
});

  3. Deletar o cookie "test cookie": 

    $('#example3').click(function(){
  var value = $.delCookie('test cookie');
  alert( "Este cookie já não existe mais \\o/");
});

Como usar

Criar/setar um cookie:

$.setCookie( 'NOME', 'VALOR', {
  duration : 1, // Em dias
  path : '',
  domain : '',
  secure : false
});

Deletar um cookie:

$.delCookie( 'NOME' );

Ler o valor de um cookie:

$.readCookie( 'NOME' );
